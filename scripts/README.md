# scripts

Helper scripts for FLEXT-API maintenance and operations using refactored classes.

All scripts in this directory follow the established patterns:
- Use ONLY refactored classes (FlextApi, FlextApiStorage, FlextApiUtilities, etc.)
- Implement FlextResult patterns for error handling
- Follow flext-core foundation patterns
- No legacy APIs, helpers, or aliases
- Hierarchical nested class usage where appropriate
